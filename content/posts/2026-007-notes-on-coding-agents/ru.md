[Подводя итоги прошлого года]{post:results-of-2025-for-me-and-the-blog} я говорил, что агенты станут главной темой 2026. Последние месяца три я постепенно разбираюсь с использованием их для разработки ПО — можно обсудить промежуточный результат.

Нормального русскоязычного термина для coding agents пока нет, поэтому я буду использовать англоязычный.

Как я использовал coding agents:

- Рефакторил и немного писал код на Rust в экспериментальном движке для игровой логики.
- Использовал их для «стандартных» задач разработки при работе над [Feeds Fun](https://github.com/Tiendil/feeds.fun) (Python).
- Как трушный хакер решил, что чтобы разобраться с чем-то, надо это что-то написать с нуля, поэтому за месяц (принципиально) навайбкодил [Donna](https://github.com/Tiendil/donna) — утилиту/окрестратор для агентов (Python).

Поскольку агенты — штука новая, то:

- никто не знает как их использовать;
- ни у кого нет разностороннего опыта их использования — либо ты выбираешь один-два подхода и работаешь свою работу ими, либо ты бесконечно экспериментируешь, не нанося пользу.

Поэтому этот пост будет в форме субъективных заметок-тезисов, без попытки объективного анализа.
