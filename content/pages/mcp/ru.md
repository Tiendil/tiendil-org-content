У этого блога есть свой MCP ([Model Context Protocol](https://ru.wikipedia.org/wiki/Model_Context_Protocol)) сервер, который вы можете подключить к своему любимому ИИ, чтобы ему и вам было удобнее читать мои посты.

Адрес MCP сервера: https://tiendil.org/mcp/

Сервер работает в экспериментальном режиме, буду благодарен за отзывы и предложения по улучшению.

Вот небольшая визуальная инструкция по подключения MCP сервера к ChatGPT: [инструкция](https://www.stephenwthomas.com/azure-integration-thoughts/how-to-enable-remote-mcp-servers-in-chatgpt-step-by-step-tutorial/).
